# FinTrack

This repository contains the FinTrack API for managing users, income, expenses, and saving goals.

How to run locally:

```powershell
npm install
cp .env.example .env         # create and populate .env with SUPABASE credentials
npm start
```

Notes:
- Keep your database credentials out of source control; use `.env`.
- This repo was extracted from an existing project. Update repository details before publishing.
